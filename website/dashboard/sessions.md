# Sessions

## Kill All Sessions

If you would like to kill all sessions simply click on the "Kill All Sessions" button.&#x20;

{% hint style="danger" %}
Please keep in mind, killing all sessions will log all of your users that have an active session out.&#x20;
{% endhint %}
